# Monkeytype-sus
## install dependency:
```
pip install flask flask_cors pyautogui
```
then 
Go to extention setting of browser -> load unpacked -> Path to "extention" folder from repo
## How to use
1. edit value of ``` wpm ``` you want in script.js in folder. (reload extension after everychange in extention setting of browser)
2. run python file.
3. Go to monkeytype site. Check output of python file. If there is POST request, si after 2s the autotyping will start.
